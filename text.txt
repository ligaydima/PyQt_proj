Проект PyQt Яндекс.Лицей
автор: Лигай Дмитрий

Тема: Оконное приложение для создания и просмотра напоминаний.
Цель: Реализовать механику приложения и обернуть в примитивную графику.
Задачи: изучить СУБД sqlite3 и библиотеку PyQt5 и применить знания в создании приложения, защитить проект.
Для того, чтобы создать напоминание, нужно нажать на кнопку «Добавить напоминание». Далее всплывет окно в котором пользователь может указать характеристики напоминания(заголовок, приоритет, дату, время и по желанию картинку).
Для просмотра напоминаний нужно выбрать категорию(«Пропущенные», «Предстоящие по времени напоминания»,  «Предстоящие по приоритету напоминания») и нажать на кнопку «Показать».
Для отображения напоминаний использовался PyQt5.QtWidgets.QVBoxLayout, а для отображения информации о каждом напоминании - PyQt5.QtWidgets.QHBoxLayout.
Для ввода информации о новом напоминании использовались поля QLineEdit (ввод заголовка), QSpinBox (ввод приоритета), QDateTimeEdit (ввод даты и времени) и QFileDialog.getOpenFileName (выбор картинки через диалоговое окно).
Все взаимодействие с базой данных перенесено в класс Interactor. Таким образом, если кто-то захочет воспользоваться механикой данного приложения, то он может просто взять все, что ему надо из этого класса (для каждой функции составлены docstring’и для того, чтобы пользователь кода знал, что делает какой кусок кода).
